# Blurfer v1.0.0

Initial public release.

## Added

- Desktop Blurfer app for sending one payload, selected payloads, or a full queue over TCP.
- Payload folder selection, visible payload list, queue ordering, per-payload delay, and per-payload port controls.
- Windows drag-and-drop copying into the selected payload folder.
- Android app with the same core workflow in a phone-friendly interface.
- GitHub release assets for Windows, Android, Linux `.bin`, and Linux AppImage.
