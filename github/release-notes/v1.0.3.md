# Blurfer v1.0.3

## Fixed

- Desktop Blurfer no longer creates a `payloads` folder beside the EXE on startup.
- Drag-and-drop now requires a selected payload folder instead of creating a fallback folder.

## Changed

- Removed the placeholder `payloads` folder from the repository.
- Windows and Linux build scripts no longer package a placeholder payload folder.
- README screenshots are organized with the desktop UI first and the Android UI shown smaller underneath it.
- Android app version bumped to `1.0.3`.

## Added

- Clean release notes for all published versions on the GitHub Releases page.
