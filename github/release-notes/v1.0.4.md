# Blurfer v1.0.4

## Changed

- Desktop UI is cleaner and easier on the eyes, with softer spacing, improved table readability, and a more compact queue control area.
- Desktop folder controls are simpler, with `Choose` and `Refresh` grouped in the top settings area.
- Payload count is now shown in the desktop payload header.
- Android UI is more compact, with smaller utility buttons and slimmer payload cards.

## Fixed

- Long Android folder names and payload names now trim cleanly instead of pushing controls around.

## Verified

- Windows EXE, Android APK, Linux `.bin`, and Linux AppImage release assets were rebuilt and published.
